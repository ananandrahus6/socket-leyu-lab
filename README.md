# socket-leyu-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定的域名或网站，而是作为通用、可独立运行的 HTML 资源集合。

## 项目简介

`socket-leyu-lab` 是一个轻量级的 HTML 页面存档库。其中包含的每个页面均为独立的 HTML 文件，可直接在浏览器中打开查看，无需额外依赖或后端服务。仓库主要服务于页面归档、测试、演示或学习参考等用途。

## 目录说明

```
.
├── index.html          # 仓库首页，列出所有归档页面链接
├── pages/              # 归档页面存放目录
│   ├── page-01.html
│   ├── page-02.html
│   └── ...
└── assets/             # 公共资源目录（可选）
    ├── css/
    └── js/
```

- **index.html**：仓库的入口页面，提供归档页面的索引列表。
- **pages/**：存放所有独立 HTML 页面的目录。每个文件是一个完整的、自包含的页面。
- **assets/**：存放共享的 CSS、JavaScript 或其他静态资源文件（如适用）。

## 页面归档说明

- 每个归档页面均为独立的 `.html` 文件，可直接在浏览器中打开。
- 页面内容不涉及任何特定网站、域名或商业服务，仅作为通用 HTML 示例或功能演示。
- 所有页面均遵循基本的 HTML 规范，可通过任何现代浏览器正常渲染。
- 若页面引用外部资源（如 CDN 库），会在页面内明确标注来源。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改归档页面。
- 欢迎通过 Issue 提出页面归档建议或报告问题。
- 不接受直接向 `pages/` 目录提交未经审核的第三方页面。
- 所有归档页面均经过基础检查，确保无恶意代码或违规内容。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `index.html` 查看所有归档页面列表。
3. 点击任意页面链接，或直接在 `pages/` 目录中打开对应的 `.html` 文件。
